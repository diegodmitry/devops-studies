# DevOps Studies

Practical DevOps projects from my learning journey, following the roadmap.sh DevOps path.

This repository documents my transition from Application Support / Production Support into Cloud and DevOps-related roles.

My goal is to build practical projects around Linux, troubleshooting, automation, networking, containers, CI/CD, cloud, monitoring, and infrastructure practices.

## Why this repository exists

I already work with production support, Linux troubleshooting, logs, tickets, databases, documentation, and Kubernetes-based environments.

This repository helps me turn that operational experience into structured DevOps practice through small, progressive, hands-on projects.
